🎯 Objetivo da atividade ex02

	Criar uma nova subclasse chamada FragTrap que herda da classe ClapTrap.

Essa nova classe é parecida com a ScavTrap da ex01, mas com atributos diferentes e uma nova 
habilidade única: highFivesGuys().

🧠 O que você aprende com essa atividade
Conceito	Aplicação prática
🧬 Herança	Como criar outra subclasse de ClapTrap
🧱 Especialização	Alterar atributos e mensagens
➕ Adição de funcionalidades	Novo método próprio da subclasse (highFivesGuys)
⚙️ Reutilização de código	Não duplicar código que já existe na superclasse
🧪 Testes em herança	Verificar ordem de construtores e métodos herdados

🔍 Pontos de atenção
Ponto crítico	O que fazer
✅ Chamar o construtor base	Use : ClapTrap(name) na inicialização de FragTrap.
✅ Ajustar os valores herdados	Redefina os atributos no corpo do construtor.
✅ Mensagens diferentes	Diferencie claramente FragTrap de ScavTrap.
✅ Não duplicar métodos	Aproveite os métodos já herdados de ClapTrap.
⚠️ Testar o novo método	Teste highFivesGuys() no main.cpp.

✅ Conclusão

A ex02 é uma continuação direta da ex01, e seu propósito é reforçar:
	Herança e construção correta de objetos.
	Especialização de classes derivadas.
	Reutilização eficiente de código com herança.
	Encapsulamento e separação de responsabilidades.
