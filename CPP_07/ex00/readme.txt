## 🧩 **Resumo da atividade ex00**

### 🎯 **Objetivo**

Implementar **funções templates genéricas** para:

1. Trocar valores (`swap`)
2. Retornar o menor entre dois valores (`min`)
3. Retornar o maior entre dois valores (`max`)

Tudo isso deve funcionar **com qualquer tipo de dado** (inteiros, floats, strings, etc), desde que os operadores necessários
 (`<`, `>`, `==`, etc) estejam definidos para aquele tipo.

---

## 📄 **O que você precisa entregar**

* `Makefile`
* `main.cpp` (com testes)
* `whatever.hpp` (onde ficam os templates)

## 🧠 **Competências desenvolvidas**

| Habilidade                      | Como é praticada na ex00                       |
| ------------------------------- | ---------------------------------------------- |
| Templates                       | Criar código que funciona com múltiplos tipos. |
| Passagem por referência/const   | Uso eficiente de memória e segurança.          |
| Comparações genéricas           | Trabalhar com operadores `<`, `>` entre tipos. |
| Organização em headers (`.hpp`) | Templates **devem estar definidos no header**. |
| Reutilização de código          | Mesmo código serve para `int`, `string`, etc.  |
